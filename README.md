# Sorgen-Rechner

Interaktiver Rechner, der die tägliche „Sorgen-Zeit" (Grübeln, Ärgern, Gedankenkarusselle) auf verschiedene Zeithorizonte hochrechnet – und sichtbar macht, wie viel Lebenszeit dabei draufgeht.

## Was macht das Tool?

- Eingabe der täglichen Sorgen-Zeit in Minuten (Slider + Eingabefeld)
- Wählbarer Zeithorizont: 1, 2, 5 oder 10 Jahre
- Ergebnis in Stunden, vollen Tagen (24h) und wachen Tagen (16h)
- Perspektiv-Box mit kontextabhängigem Vergleich

## Einbettung

Das Tool ist als einzelne `index.html` gebaut (kein Build-Prozess, keine Abhängigkeiten) und wird über GitHub Pages gehostet. Der Link kann direkt in Notion eingebettet werden.

**Live:** `https://jubuekal-source.github.io/sorgen-rechner/`
